# Impeccable Design Language

Adopted from https://github.com/pbakaus/impeccable

## Core Principles
- **Typography Over Decoration**: Use bold, deliberate typography rather than excessive containers or borders.
- **High Contrast**: Ensure clear readability and visual hierarchy through contrast.
- **Intentional Color**: Use colors for purpose, not just aesthetics (e.g., Orange for Action).
- **Spatial Design**: Use whitespace to group related elements and provide "breathing room".

## Anti-Patterns to Avoid
- Pure black/gray (use tinted shades).
- Excessive card nesting.
- Generic system fonts.
- Low-contrast text.

## Commands (AI Skill)
- `/audit`: Review the current UI for quality and consistency.
- `/normalize`: Align elements with the design system.
- `/polish`: Final pass for visual excellence.
- `/critique`: UX-focused review.
